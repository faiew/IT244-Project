# Smart Clinic Database System

## Project Overview

The Smart Clinic Database System is a relational database project created as part of a university database course. The project aims to improve the management of a private clinic by replacing manual record-keeping with a structured database system. It stores and manages patient information, doctor details, appointments, treatments, medicines, and payments using MySQL. The project covers database design, implementation, and SQL operations to support efficient data management and information retrieval.
